# BUT1 Portfolio – Preuves et Travaux

## 📌 Projet : Data Cleaning (SAE)
Ce projet démontre plusieurs apprentissages critiques :
- AC12.02 : Comprendre qu'une analyse correcte nécessite des données propres
- AC11.03 : Maîtriser la syntaxe Python
- AC12.03 : Utiliser des synthèses graphiques pour décrire une variable

### 🔧 Outils utilisés
- Python (pandas, numpy)
- Jupyter Notebook
- Matplotlib / Seaborn

### 📊 Description du travail
J'ai nettoyé un dataset contenant des valeurs manquantes, des doublons et des incohérences.
Les étapes principales :
1. Détection des valeurs manquantes
2. Normalisation des colonnes
3. Visualisation des distributions
4. Export du dataset propre

### 📁 Fichiers inclus
- `cleaning.ipynb` : Notebook principal
- `graph.png` : Visualisation produite
- `data_cleaned.csv` : Données finales

### 🎯 Compétences démontrées
- Compréhension du besoin (AC11.01)
- Nettoyage de données (AC12.02)
- Visualisation (AC12.03)
- Communication des résultats (AC13.04)
